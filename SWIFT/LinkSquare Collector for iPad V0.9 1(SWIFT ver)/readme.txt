
이 프로젝트는 LinkSquare 어플리케이션 스위프트 버전입니다.

swift 언어의 튜토리얼은 같은 디렉토리의 "swift 자료.pdf"를 

참고하시면 됩니다.

기존에 objective-C를 목적으로 작성된 API는 wrapping 작업을 통해서

사용이 가능합니다.

>>> C lang(API) -> obj-c -> bridge header(wrapping) -> swift

image 폴더는 어플 내에서 사용한 asset들이 들어있습니다.

test 폴더들은 사용하지 않습니다.

*cocoa pod(오픈소스 라이브러리 도구)를 사용하기 위해서는 프로젝트 파일 아닌
워크스페이스 파일을 열어주어야합니다.
(이것도 역시 튜토리얼 파일에 있습니다.)

이외의 참고사항은 같은 디렉토리의 "LSCollector_iOS_documents_20180131"폴더의
자료들을 확인하거나 소스 내의 주석을 참조하시면 됩니다.
